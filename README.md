# CodSoft Machine Learning Internship Portfolio

This repository contains the projects I completed during my Machine Learning internship at CodSoft. Each project demonstrates the application of data preprocessing, exploratory data analysis (EDA), predictive modeling, and evaluation techniques.

---

## 🛠️ Internship Projects

### 1. 🎬 Movie Rating Prediction
*   **File:** `Movie Rating prediction.ipynb`
*   **Objective:** Build a model that predicts the rating of a movie based on features like genre, director, and actors.
*   **Key Skills:** Data cleaning, regression analysis, feature engineering, and handling categorical variables.

### 2. 🚢 Titanic Survival Prediction
*   **File:** `TITANIC.ipynb`
*   **Objective:** Create a classic classification system to predict whether a passenger would survive the Titanic shipwreck based on demographics, socio-economic status, and cabin details.
*   **Key Skills:** Exploratory Data Analysis (EDA), handling missing data, binary classification modeling, and performance evaluation metrics.

### 3. 🪻 Iris Flower Classification
*   **File:** `IRIS.ipynb`
*   **Objective:** Optimize a machine learning model to classify different species of Iris flowers (Setosa, Versicolor, Virginica) based on sepal and petal measurements.
*   **Key Skills:** Supervised learning, multi-class classification, data visualization, and model hyperparameter tuning.

### 4. 🚫 SMS Spam Detection
*   **File:** `spam detection.py`
*   **Objective:** Build an AI-driven text classification pipeline to accurately identify whether an incoming SMS message is "Spam" or "Ham" (Legitimate).
*   **Key Skills:** Natural Language Processing (NLP), text preprocessing (tokenization, stop-word removal), TF-IDF vectorization, and Naive Bayes / NLP classification models.
